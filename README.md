# FollowMeDrone: Autonomous Follow-Me Drone with Tracking and Gesture Media Control 

This repository contains the code for our EECS 206A Final Project: a DJI Tello drone that can autonomously follow a person and respond to gesture commands.

## Requirements  
- **DJI Tello Drone**  
- **Teensy 4.1** microcontroller  
- **2x MPU6050 IMU sensors**  

## Usage  
1. Start gesture recognition:  
   ```bash  
   python3 gesture_recognition.py
2. Start the main control loop for face tracking:
   ```bash  
   python3 face_tracking.py
